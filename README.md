# 文件操作案例

工作中常常遇到的几个问题：文件上传、文件下载、文件批量上传、服务器文件夹打包，下载、文件压缩上传、根据内容，生成二维码等等问题，所以我这里直接把这些常用方法统一写一个综合案例。

### 文件操作

- html上传：文件上传
- html上传：多文件上传
- html上传：图片上传 - 压缩
- ajax上传：FormData上传
- 其他：将指定文件夹打包为zip
- 其他：生成二维码
- 下载文件

通过案例，让自己对于文件的操作更加的了解，以及工作中使用起来更加的方便。
